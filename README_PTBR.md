# LINEOS — Plataforma de Zootecnia de Precisão

![LINEOS Premium Login](public/login_background.png)

A LINEOS é uma plataforma digital de espectro total, projetada para criadores de cães e gatos de alta performance. Integra genética avançada, gestão financeira e monitoramento de saúde em uma experiência premium e fluida.

## 🚀 Funcionalidades

### 🧬 Inteligência Genética
- **Simulador de Endogamia**: Calculadora de COI e impacto de ROI projetado via custos de depressão genética.
- **Gestão de Pedigrees**: Linhagens maternas e paternas rastreadas com análise automatizada de parentesco.
- **Potencial Genético**: Avaliação de EBVs para seleção zootécnica.

### 💰 Ecossistema Financeiro
- **ROI por Ninhada**: Análise de retorno para cada projeto de acasalamento.
- **DRE Especializado**: Contabilidade setorial para criação animal.
- **Análise de Custos**: Rastreamento de manutenção, saúde e exposições.

### 🏥 Saúde & Manejo
- **Nutrição de Precisão**: Cálculos baseados no guia FEDIAF 2025.
- **Histórico Clínico**: Vacinas, exames e tratamentos centralizados.
- **Manejo Mobile**: Interface otimizada para campo.

## 🛠️ Stack Tecnológica

- **Frontend**: React 18, TypeScript, Vite.
- **Estilo**: Tailwind CSS, Shadcn UI, Lucide icons.
- **Estado**: Zustand, React Query.
- **Backend**: Supabase (PostgreSQL, Auth, RLS).

## 🔐 Segurança

Este projeto segue as melhores práticas de **Row-Level Security (RLS)**. Todos os seus dados são isolados por ID de usuário, garantindo confidencialidade e integridade máxima dentro do ecossistema.
